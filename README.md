# 🌡️ Weather Temperature Predictor

A professional, interactive machine learning web application built with **Streamlit** that predicts **Maximum Temperature** using a **Linear Regression** model.

---

## 🚀 Live Demo

👉 **[Click here to view the app](https://temperature-prediction-linear-regression.streamlit.app/)**

---

## ✨ Key Features

- **Real-time Prediction**: Adjust weather parameters (Min Temp, Rainfall, Humidity, etc.) using interactive sliders
- **Premium UI**: Custom adaptive theme with a formal color palette (`#054A91`, `#3E7CB1`, `#DBE4EE`, `#F17300`)
- **Glassmorphism Design**: Modern prediction cards with transparent glass effects and responsive layout
- **Model Insights**: Displays live model accuracy and dataset summary

## 🛠️ Tech Stack

- **Python** (Core Language)
- **Scikit-learn** (Linear Regression Model)
- **Streamlit** (Frontend Framework)
- **Pandas & Numpy** (Data Processing)
- **GitHub & Streamlit Cloud** (Deployment)

---

## 📊 Dataset Overview

The project uses **Weather_Data.csv**, featuring:

**Predictors**: MinTemp, Rainfall, Humidity3pm, Pressure3pm, RainToday  
**Target**: MaxTemp (Maximum Temperature)

---

## ⚙️ How to Run Locally

1. **Clone the repo**  
   ```bash
   git clone https://github.com/pro382r/Temperature-prediction.git
   ```
2. **Install requirements**
   ```bash
   pip install -r requirements.txt
   ```
3. **Launch the app**
   ```bash
   streamlit run app.py
   ```
